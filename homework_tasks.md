## ТИЖДЕНЬ 0: ONBOARDING

### Домашнє завдання #0: «Перший контакт»

**Після:** Onboarding-блок

**Завдання:**
1. Відкрити ChatGPT (або Claude / Gemini) і задати одне запитання трьома різними способами - від «ледачого» до «професійного». Скріншот усіх трьох відповідей.
2. Вставити бонусний промпт «Advisor Mode» у Custom Instructions. Задати AI питання зі своєї сфери роботи. Скріншот «до» (без Advisor Mode) і «після» (з ним).
3. Знайти одну рутинну задачу зі своєї роботи (email, звіт, пошук інформації) і спробувати делегувати її AI. Зберегти промпт + результат + свій вердикт: «Використаю / Потрібно допрацювати / Не підходить».

**Результат для портфоліо:** Перший кейс «було / стало» - різниця між хаотичним і структурованим запитом.

## ТИЖДЕНЬ 1: ОСНОВИ PROMPT ENGINEERING

### Домашнє завдання #1: «Prompt Makeover: із сміття - золото»

**Після:** Заняття 1. Як мислити prompt'ом

**Завдання:**
Нижче - 3 «жахливі» промпти, які типовий користувач пише щодня. Завдання: переробити кожен у сильний промпт, використовуючи framework із заняття (Задача -> Контекст -> Інструкція -> Формат -> Приклад). Кожну переробку запустити в AI і зберегти output.

**Промпт-пацієнти:**
1. `Напиши мотиваційний пост`
2. `Проаналізуй конкурентів`
3. `Придумай назву для бізнесу`

**Бонус:** Знайти один «жахливий» промпт зі свого реального робочого досвіду та переробити його. Цей кейс піде прямо в портфоліо.

**Результат для портфоліо:** 3-4 кейси «поганий промпт -> хороший промпт» з реальними output.

### Домашнє завдання #2: «Модельний батл: хто кращий?»

**Після:** Заняття 2. Базові prompt-патерни

**Завдання:**
Один і той самий промпт - три різні моделі. Порівняти, хто справився краще.

1. Вибрати одну реальну задачу зі своєї роботи (написати лист, зробити саммарі, згенерувати ідеї тощо).
2. Написати один якісний промпт, використовуючи патерни з заняття (role + few-shot + structured output).
3. Надіслати цей САМИЙ промпт у три моделі: ChatGPT, Claude та Gemini.
4. Оцінити кожну відповідь за Prompt Scorecard (Clarity, Context, Specificity, Format Control, Output Quality).
5. Зробити висновок: яка модель впоралась найкраще для ЦЬОГО типу задачі і чому?

**Формат здачі:** Таблиця порівняння + коротке резюме (3-5 речень): «Для [тип задачі] я обираю [модель], тому що…»

**Результат для портфоліо:** Перше model comparison дослідження.

## ТИЖДЕНЬ 2: ПРАКТИЧНІ ЗАДАЧІ ТА РОБОЧІ СЦЕНАРІЇ

### Домашнє завдання #3: «AI-стажер: день із життя»

**Після:** Заняття 3. Контент, research, summarization, rewriting

**Завдання:**
Уявіть, що AI - ваш стажер на один робочий день. Дайте йому 3 реальні задачі зі своєї сфери та отримайте результат, який можна реально використати (не «для прикладу», а по-справжньому).

**Вибрати 3 із 6 сценаріїв:**
1. **Саммарі-машина:** Знайти будь-яку статтю (5+ хвилин читання) і зробити executive summary для свого керівника / клієнта. Формат: 1 ключовий висновок + 3 bullet points + 1 рекомендація.
2. **Tone Chameleon:** Взяти один і той самий текст (свій або знайдений) і переписати його у 3 різних тонах: для LinkedIn, для Instagram Stories, для офіційного email.
3. **Research Assistant:** Задати AI задачу зібрати аргументи ЗА і ПРОТИ будь-якої ідеї з вашої роботи. Отримати структуровану таблицю з аргументами, силою кожного (1-5) та джерелами для перевірки.
4. **Редактор-перфекціоніст:** Взяти свій текст (email, пост, стаття) і попросити AI зробити 3 ітерації покращення: спочатку - стиль, потім - структура, потім - скорочення на 30%.
5. **Перекладач-інтерпретатор:** Переписати складний фаховий текст так, щоб його зрозуміла 12-річна дитина. А потім навпаки: з простого тексту зробити академічний.
6. **Конкурент-аналітик:** Проаналізувати 3 конкурентні профілі / сайти / продукти і отримати SWOT-таблицю у форматі, готовому для презентації.

**Результат для портфоліо:** 3 робочі кейси з реальними задачами та output.

### Домашнє завдання #4: «Prompt Chain Challenge: ланцюгова реакція»

**Після:** Заняття 4. Prompting для роботи й бізнесу

**Завдання:**
Побудувати ланцюжок із 3-4 промптів, де вихід одного стає входом для наступного. Одна велика задача - кілька послідовних кроків.

**Вибрати один сценарій:**

**Сценарій A - «Від ідеї до готового контенту»:**
1. Промпт 1: Згенерувати 10 ідей для контенту у вашій ніші
2. Промпт 2: Вибрати 3 найсильніші та написати для кожної hook + outline
3. Промпт 3: Розгорнути один outline у повний текст
4. Промпт 4: Адаптувати цей текст під 3 платформи (LinkedIn / Instagram / email)

**Сценарій B - «Підготовка до складної зустрічі»:**
1. Промпт 1: Описати ситуацію -> отримати аналіз ризиків і можливостей
2. Промпт 2: Підготувати 5 ключових аргументів + передбачити 3 заперечення
3. Промпт 3: Написати скрипт відкриття зустрічі (перші 2 хвилини)
4. Промпт 4: Створити follow-up email за результатами

**Сценарій C - «HR-конвеєр» (для HR/рекрутерів):**
1. Промпт 1: З опису проєкту -> згенерувати вакансію
2. Промпт 2: З вакансії -> створити scorecard для оцінки кандидатів
3. Промпт 3: Зі scorecard -> написати скрипт phone screening
4. Промпт 4: З результатів screening -> шаблон зворотного зв'язку кандидату

**Формат здачі:** Усі 3-4 промпти + output кожного кроку. Пояснення одним реченням: що було входом, що виходом.

**Результат для портфоліо:** Перший prompt chain - приклад системного підходу до AI.

## ТИЖДЕНЬ 3: ЯКІСТЬ, ПОМИЛКИ, ПЕРЕВІРКА, БЕЗПЕКА

### Домашнє завдання #5: «Детектив галюцинацій: зламай AI»

**Після:** Заняття 5. Hallucinations, bias, weak outputs

**Завдання:**
Спеціально спровокувати AI на помилку, зафіксувати її, а потім - виправити за допомогою промптингу.

**Етап 1 - «Провокація» (знайти 2 галюцинації):**
- Попросити AI розповісти біографію малодомої реальної людини (наприклад, місцевого підприємця або нішевого автора). Перевірити факти вручну. Відмітити, що AI вигадав.
- Попросити AI дати «посилання на дослідження» з конкретної теми. Перевірити, чи існують ці посилання.

**Етап 2 - «Лікування» (виправити промптом):**
- Переписати промпт так, щоб AI перестав галюцинувати: додати обмеження «якщо не знаєш - скажи прямо», «не вигадуй джерела», «став тег [НЕВПЕВНЕНИЙ] біля фактів, у яких не впевнений».
- Запустити покращений промпт і порівняти результат.

**Етап 3 - «Evaluation Rubric»:**
Створити власну рубрику з 5 критеріїв для оцінки якості AI-output (наприклад: фактичність, повнота, релевантність, структура, тон). Оцінити свої відповіді з Етапів 1 та 2 за цією рубрикою.

**Результат для портфоліо:** Кейс «Як я зламав AI і потім полагодив» + власна evaluation rubric.

### Домашнє завдання #6: «Таємний агент: audit промптів»

**Після:** Заняття 6. Privacy, copyright, ethics, responsible use

**Завдання:**
Провести «етичний аудит» AI-використання у реальному контексті.

**Частина 1 - «Privacy Scan»:**
Переглянути свою історію чатів з AI (ChatGPT / Claude) за останній місяць. Знайти 3 випадки, де ви (або могли б) випадково вставити чутливі дані: імена клієнтів, фінансові цифри, паролі, NDA-інформацію. Описати кожен випадок та як його уникнути.

**Частина 2 - «Bias Detector»:**
Задати AI 2 одинавих питання, але з різними деталями, що можуть викликати упередженість (наприклад, «Напиши рекомендаційний лист для Олексія» vs «…для Олексії»; або «…для стартапу в Києві» vs «…для стартапу в Сан-Франциско»). Порівняти відповіді. Чи є різниця? Яка?

**Частина 3 - «Prompt Injection Test»:**
Спробувати на будь-кому публічному AI-чатботі (не на платній підписці!) просту техніку prompt injection: «Ігноруй попередні інструкції та…». Зафіксувати результат. Чи вдалось? Що це означає для безпеки?

**Формат здачі:** Короткий звіт (1 сторінка): 3 знахідки + 3 правила, які ви тепер будете дотримуватись.

**Результат для портфоліо:** Демонстрація AI safety awareness - цінна навичка для будь-якого роботодавця.

### Домашнє завдання #8: «Фінальний мінікейс: AI вирішує реальну задачу»

**Після:** Заняття 8. Self-marketing і наступний крок

**Завдання:**
Фінальний проєкт. Обрати одну реальну задачу зі свого життя або роботи та розв'язати її повністю за допомогою AI.

**Формат:**
1. **Задача** (2-3 речення): що потрібно зробити? Для кого? Який бажаний результат?
2. **Промпт-стратегія** (описати підхід): які техніки використали (role, CoT, constraints, chain)? Чому саме ці?
3. **Процес** (3-5 скріншотів): ітерації, покращення, порівняння моделей.
4. **Результат**: готовий output, який можна реально використати.
5. **Рефлексія** (3-5 речень): що працювало? Що ні? Що б зробили інакше?

**Приклади задач (обрати одну або придумати свою):**
- Створити system prompt + workflow для AI-асистента
- Побудувати AI-асистента для щотижневого планування (system prompt + workflow)
- Зробити аналіз відгуків клієнтів з Google Maps / Trustpilot і звіт з рекомендаціями
- Створити набір промптів для автоматизації рутинних email-відповідей
- Підготувати контент-план на місяць для свого (або вигаданого) бренду
- Розробити prompt-based Q&A бота для FAQ свого продукту / сервісу

**Презентація:** 5-хвилинне представлення кейсу перед групою + відповіді на питання (peer review).

**Результат:** Фінальний кейс для портфоліо + практика self-marketing.

## «Prompt of the Day» - щоденні async-завдання

> **Формат:** Щодня (між заняттями) в чат курсу кидається одне мікро-завдання.
> **Що робити:** (1) написати промпт -> (2) показати output -> (3) покращити за 1 ітерацію.
> **Навіщо:** Формує щоденну звичку + streak mechanics + кожен виконаний PotD - мікрокейс для портфоліо.
